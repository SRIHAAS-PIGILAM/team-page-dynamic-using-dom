
# Team Page - Dynamic UI using DOM

This project is a dynamic "Meet the Team" page created as part of the Crio.do curriculum. It demonstrates how to use JavaScript to manipulate the Document Object Model (DOM) to build a user interface based on data.

Instead of writing static HTML for every team member, this project uses a JavaScript "factory" approach. It takes a `team` array of objects and converts them into styled HTML cards automatically.

### Key Learnings:
* **DOM Creation**: Using `document.createElement` to generate sections and text elements.
* **Attribute Management**: Setting `src`, `alt`, and `id` dynamically.
* **Responsive Layouts**: Using CSS Flexbox and Media Queries to ensure the site looks great on mobile, tablet, and desktop.
